## 1. Vision (Foundation)

**7 Days to Automate** is an automation-focused overhaul mod for *7 Days to Die*, inspired by:

* Minecraft modpacks (SkyFactory, Tekkit, Applied Energistics 2)
* Factory-style games (factorio, Satisfactory)
* Long-term automation, logistics, and research

Core goals:

* Automation that feels **vanilla-adjacent**, not tacked on
* Deep progression and complexity over time
* Systems that scale to **large factories** without performance collapse
* Data-driven, XML-friendly, and mod-compatible

## 6. Higher Logic Registry (HLR) — **Current Focus**

### 6.1 What HLR Is

HLR is a **world-scoped simulation layer** that:

* Runs alongside the world (not above it)
* Only simulates machines that **cannot simulate themselves**
* Handles unloaded chunks safely

Key rule:

> If a block is loaded, it does its own logic
> If it unloads, it hands off to HLR

---

### 6.2 What HLR Does

* Simulates machines in unloaded chunks
* Simulates controllers if they unload
* Simulates power flow for unloaded networks
* Prevents production loss or duplication
* Guarantees deterministic results

---

### 6.3 HLR Lifecycle (Defined)

1. Block is active → simulates itself
2. Chunk unload imminent:

   * Machine packages its state
   * Registers snapshot with HLR
   * Sets `simulation = true`
3. HLR simulates at fixed interval (~1 second)
4. Chunk loads:

   * Machine pulls state from HLR
   * Applies updates
   * Unregisters from HLR
   * Resumes self-simulation

---

### 6.4 Performance Target (Locked In)

* **5,000 machines** simulated concurrently
* Tick rate: ~1 update per second
* No noticeable impact on gameplay

---

### 6.5 Technical Decisions (Completed)

* HLR tied to **World**, not static
* Attached via `ConditionalWeakTable<World, HLR>`
* Instantiated on world creation
* Cleaned up on world shutdown
* Uses GUIDs for machine identity
* HLR will have its **own save data**

Status:
✔ HLR instantiation complete
✔ World lifecycle hooked
🟡 Save/load design in progress
🟡 Machine registration not implemented yet
